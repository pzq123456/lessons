# 遥感地学分析 quiz
- 第一讲 绪论
- 名词解释
    - 遥感定义: 
      - 从远离地面的不同工作平台上（飞机卫星）
      - 通过传感器对地球表面的电磁波信息进行探测
      - 并经信息的传输、处理判读分析
      - 实现研究地物空间形状位置性质及与环境间相互关系的一门应用技术学科
    - 遥感地学分析的概念:
      - 以地学规律为基础，对遥感信息进行的分析处理过程，是地学分析方法与遥感图像处理方法的有机结合
      - 一方面可提高地学研究的视域提高区域认知水平
      - 另一方面可以改善遥感分析处理识别的精度
    - 反射率: 反射能量与入射总能量之比
    - 透射率: 入射光透过地物的能量与总入射能量之比
    - 地物反射光谱: 地物反射率对入射光波长的变化规律
    - 黑体: 理想状态下，任何温度都对入射能量完全吸收
    - 发射率: 地物发射能量与同等状态下黑体发射能量之比

    - 如何理解传感器的辐射分辨率、光谱分辨率、空间分辨率、时间分辨率？
      - 辐射分辨率：传感器对辐射强度的分辨能力
      - 光谱分辨率：选取的波段多少、范围及宽窄所确定
      - 空间分辨率：最小能分辨的地物大小或像元吃尺寸
      - 时间分辨率： 重访周期，重新观测同一地点的频率

    - 列举几种典型地物反射光谱特征
      - 岩石：岩石的反射光谱特征是地质遥感的基础。矿物含量、风化程度含水量等都会影响
      - 土壤：反射率曲线无明显起伏。越细越亮，含水量有机质含量越高越暗。与肥力也有关系
      - 水体：反射主要在蓝光波段，其他波段吸收为主，红外波段后就是一个吸收体
      - 植被：植被的反射波谱曲线明显而独特，主要分为三个波段
        - 可见光04-076：0.55 绿色附近有一小反射峰 0.45蓝色和0.67红色是两吸收谷
        - 近红外07-08：有一反射陡坡 1.11附近有一反射峰
        - 中红外13-25：受绿色植物含水量的影响 吸收率大增反射率大幅度减小 1.45 1.95 2.7 为中心的是水的吸收带
    - 什么是黑体？
      - 理想的辐射体指任何温度下对各种波长的电磁波全部吸收
    - 根据发射率与波长的关系,简述地物类型的划分依据？
      - 黑体发射率为1，不随波长变化
      - 灰体发射率波长小于1，不随波长变化
      - 选择性辐射体：发射率小于1，随波长变化
    - 简述目视解译的概念,并说出目视解译与计算机解译之间的区别.
      - 目视解译：肉眼或借助简单的工具，直接由人来识别图像的特征提取有用的信息，即人把物体与图像联系起来的过程
        - pros:能综合运用地学知识判断地学现象
        - cons:速度慢效率低处理量小
      - 计算机解译：速度快效率高但是无法综合运用地学知识判别特定地学现象
    - 什么是解译标志？
    - 指能够反映和表示地物信息的 遥感影像的各种特征。这些特征能够帮助判读者识别图上的各种现象或目标。分为直接判读标志和间接判读标志。
    - 有哪些解译标志,这些解译标志中哪些是直接解译标志,哪些是间接解译标志
    - 直接解译标志: 
      - 色调、识别地物的基本依据，属性差异遥感图像上表现为色调差异，灰度级16人眼局限性
      - 颜色、对不同波段反射的电磁波能量的差异，也是遥感图像解译的基本标志
      - 形状、外部轮廓，不同目标的顶视平面存在差异
      - 大小、遥感图像上形状面积体积的度量关系，重要数量特征也可以推断地物属性
      - 纹理、目标地物内部色调有规则变化造成的影像结构
      - 阴影、电磁辐射被遮挡产生能量减弱、受辐射入射角影响、据此推测高度及性质
      - 图形、地物有规则排列而成的图形结构
    - 间接解译标志: 
      - 位置、指目标地物分布的地点
        - 任何地物与其周围地理环境或生态环境都存在一定的空间关系并受到周围环境的制约
      - 相关布局 多个目标地物见的空间配置关系
